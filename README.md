# Flash Diag 360° — Équipementiers automobile

Site marketing BearingPoint pour l'offre **Flash Diag 360°** dédiée aux équipementiers automobile.

## Aperçu

5 semaines. Un diagnostic industriel augmenté par l'IA. Une trajectoire de productivité activable
sans investissement lourd dans les standards OEM (IATF 16949, APQP, PPAP).

## Stack

Site statique pur — HTML / CSS / JS, aucune dépendance externe.

- `index.html` — structure
- `styles.css` — styles BearingPoint (palette signature red, font Aptos)
- `script.js` — interactions, démo vidéo synthétique (Web Audio), animations
- `assets/` — logo et images BP

## Lancer en local

```bash
python -m http.server 8765
```

Puis ouvrir [http://127.0.0.1:8765](http://127.0.0.1:8765).
